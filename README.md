# Gava

Unit test helpers for DOM libraries

## Contribution

### Prepare

1. Clone project

```bash
git clone https://github.com/VSymonenko/gava.git
```

2. Install Node.js dependencies

```bash
npm i
```

## Release

1. Run release locally

```bash
npx tsx release.mts -d false
```
